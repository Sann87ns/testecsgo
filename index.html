Seus primeiro passo no Country Strike: Global Offensive

Acabou de baixar csgo para jogar e quer começar com o pé direito no game, veja abaixo os primeiros passo que você deve seguir

Comandos para treino

bot_kick : Para kickar os bots da partida

bot_add_ct : Adiciona um bot no lado CT

bot_add_tr : Adiciona um bot no lado TR

bot_dont_shoot 1 : Os bots param de atirar

bot_stop 1 : Os bots ficam parados

bot_crouch 1 : Os bots ficam agachados

bot_place : Posiciona um bot no local onde você está mirando

mp_limitteams 0 : Ilimita a quantidade de bots em um determinado time

mp_autoteambalance 0 : Desativa o equilíbrio das equipes, permitindo assim colocar os 5 bots contra 1 player

sv_cheats 1 : Para liberar alguns comandos

mp_roundime_defuse 60 : Para aumentar o tempo da partida

mp_buytime 60000 : Para aumentar o tempo de compra

mp_startmoney 60000 : Para aumentar o dinheiro no início da partida

mp_buy_anywhere 1 : Para poder fazer compras de qualquer parte do mapa

mp_restartgame 1 : Para resetar a partida e ativar os comandos utilizados

sv_infinite_ammo 1 : Para ter munição infinita

sv_showimpacts 1 : Para mostrar onde foi o tiro

sv_grenade_trajectory 1 : Para mostrar a trajetória da granada

sv_grenade_trajectory_time 4 : Diminui a visualização da trajetória da granada para 4 segundos

noclip : Para "voar" pelo mapa
